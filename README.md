Trip Planner Local — это легковесное, быстрое и полностью приватное веб-приложение (SPA) для планирования самостоятельных путешествий. 

В отличие от тяжелых облачных сервисов, это приложение хранит все данные исключительно на вашем устройстве (в localStorage браузера). Никаких регистраций, подписок и скрытой синхронизации.

## Главные возможности

- Интерактивная карта: Интеграция с Leaflet. Кликайте по карте для получения точных координат, а приложение само нарисует нить вашего маршрута.
- Умный таймлайн (Drag & Drop): Интерактивная сетка расписания с функцией масштабирования (Zoom). Перетаскивайте карточки из «Банка идей» прямо в расписание нужного дня.
- Учет логистики: Автоматическая отрисовка заложенного времени на дорогу между событиями. Защита от накладок в расписании (конфликты времени подсвечиваются).
- Финансовый трекер: Задавайте общий бюджет поездки и фиксируйте расходы по категориям. Приложение автоматически пересчитает остаток.
- PWA и Офлайн: Работает без подключения к интернету. Легко устанавливается на главный экран смартфона (iPhone/Android) как нативное приложение.
- Ультимативный экспорт:
  - Резервное копирование всех данных в JSON.
  - Импорт бронирований из стандартых календарей (.ics).
  - Сохранение маршрута и сводки бюджета в красивый PDF.
  - Быстрое копирование расписания на день в формате текста (идеально для Telegram/WhatsApp).

## Как использовать (Установка на смартфон)

Поскольку проект работает полностью на стороне клиента, вы можете использовать его прямо из браузера:
[Открыть приложение](https://dewinter5.github.io/trip-planner-site/))

Для установки на iPhone (iOS):
1. Откройте ссылку в браузере Safari.
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой внизу экрана).
3. Выберите «На экран "Домой"» (Add to Home Screen).
4. Запускайте планировщик с рабочего стола как обычное приложение.

## Безопасность данных

Все ваши планы, координаты и бюджеты хранятся локально в кэше вашего браузера. 

> Внимание: Если вы очистите историю или данные сайтов в браузере, либо удалите ярлык приложения с телефона, данные будут стерты. 

Регулярно используйте кнопку «Экспорт JSON» для создания резервных копий (бэкапов) вашей поездки перед закрытием браузера или сменой устройства.

## Технологический стек

- HTML5 / CSS3
- JavaScript (Vanilla) — логика работы с данными и DOM.
- Tailwind CSS (через CDN) — современный UI/UX дизайн.
- Leaflet.js — картография и маршрутизация.
- html2pdf.js — генерация PDF-отчетов прямо в браузере.
